# ShopNow – Fashion Shopping App

A Myntra-inspired fashion shopping single-page application built with HTML, CSS and Vanilla JavaScript.

## 🌐 Live Demo
[https://rajabaniya786.github.io/shopnow-fashion-app](https://rajabaniya786.github.io/shopnow-fashion-app)

## 🎨 Figma Design
[View Design on Figma](https://www.figma.com/design/rxt5jeFyzUo0Vsm4dkWtu4)

## ✨ Features
- 🛍️ Browse 16 dummy fashion products across 6 categories
- 🔍 Live search by product name or brand
- 🎛️ Multi-filter sidebar (Category, Brand, Price, Size, Discount, Rating)
- ↕️ Sort by Recommended, New, Popularity, Price, Discount
- ❤️ Wishlist toggle per product
- 🛒 Add to Bag with live counter
- 📱 Responsive grid layout
- 🎉 Toast notifications

## 🛠️ Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (DM Sans + Playfair Display)
- GitHub Pages (deployment)
- Figma (design)

## 🚀 Run Locally
Just open `index.html` in your browser — no build step required!
